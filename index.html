Introduction

JavaScript is cross platform, object oriented scripting language

JavaScript is a programming language that makes websites interactive and dynamic. With JavaScript, you can make a webpage respond to user actions, like clicking buttons or entering information, and update content without needing to refresh the whole page.

It's used to create things like pop-up messages, interactive forms, and animations on webpages.

It's a fundamental technology for modern web development and adds life and functionality to websites.

Variables

a variable is a named container that can hold various types of data, such as numbers, strings, or objects.

Variables are used to store and manage information in your code. You can assign values to variables, update those values, and use the

variable's name to access the stored data, making your JavaScript programs dynamic and flexible.

Declearing Variable

  • var (Variable):

    • Variables declared with var are function-scoped, which means they are only accessible within the function they are defined in. If declared outside of any function, they are globally scoped.

    They can be re-declared within the same scope without any error.

  • let (Block-Scoped Variable):

    • Variables declared with let are block-scoped, which means they are only accessible within the block in which they are defined.

    They can be updated but not re-declared in the same scope.

  • const (Constant):

    • Variables declared with const are also block-scoped like let.

    However, they cannot be re-assigned once they are given a value. They are constant.

    const z = 5; z = 10; const z = 20;

Variable Scope

  • Global Scope:

    • Variables declared outside of any function or code block have global scope.

    Global variables are accessible from anywhere in your JavaScript code, including inside functions.

    Be cautious with global variables as they can lead to unintended side effects and make your code harder to maintain.

  • Local Scope

Variables declared within a function have local scope, which means they are only accessible within that function.

Local variables are isolated from the rest of the code, reducing the risk of unintended changes or conflicts.

    • Java is a general-purpose programming language that can be used for various purposes, including web development (Java EE), desktop application development (Java SE), mobile app development (Android), and server-side development (Java Spring). It is known for its platform independence and "write once, run anywhere" philosophy.

    Java has a more strict and statically-typed syntax. Variable types must be declared explicitly, and type checking is done at compile-time.

    Java applications are compiled into bytecode, which is executed by the Java Virtual Machine (JVM). Java can run on various platforms as long as there is a compatible JVM available.

    Java also has a large community and is widely used in enterprise-level applications. It has a rich ecosystem of libraries and frameworks (e.g., Spring, Hibernate).
